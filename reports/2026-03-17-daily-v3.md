## 二、项目数据发现

### 2.1 价格

**观察到的事实** [证据层级: A] [数据截至: 2026-03-17]

- WTI 近5日：$87.25 → $95.73 → $98.71 → $93.50 → $97.90（日波动幅度5-10%）
- Brent 近5日：$91.98 → $100.46 → $103.14 → $100.21 → $104.31
- WTI-Brent 价差近5日：-$4.73 → -$4.73 → -$4.43 → -$6.71 → -$6.41（Brent溢价趋势走阔）

**与这些事实一致的解释**

- 解释1：Brent溢价走阔与国际海运原油供给受阻一致——Brent定价的海运原油比管道输送为主的WTI受影响更大
- 解释2：价差波动也可能反映短期流动性差异、基金重新配置、或区域性炼厂需求变化

**当前不能确认的事项**

- 不能仅从价差变化确认中东供给中断的精确规模
- WTI-Brent价差受多因素驱动（管道运力、库欣库存、出口节奏等），不能简单归因于单一原因

**反证/削弱该判断的数据**

- 如果WTI-Brent价差快速收窄至-$3以下，则国际海运原油受扰的信号减弱
- 如果WTI涨幅追上Brent，可能反映北美供给也开始受限（不同叙事）

### 2.2 期货曲线

**观察到的事实** [证据层级: A] [数据截至: 2026-03-17]

- 近月（2026-04）：$97.88 → 远月（2027-03）：$72.94
- M1-M2价差：+$0.81（近月溢价）
- M1-M6价差：+$14.99（半年价差近$15）
- 12合约全线Backwardation，远月逐月递减
- 远月价格分布：$97.88 → $97.07 → $92.73 → ... → $72.94

**与这些事实一致的解释**

- 解释1：深度Backwardation与短期供应紧张预期一致——现货/近月溢价高企
- 解释2：远月价格（$73-$77区间）可能更接近市场对中期基本面均衡水平的估价，但也可能包含流动性折价和风险贴水
- 解释3：当前Backwardation与STEO显示的全球供需过剩（+2.56 mb/d）并存，两者反映不同时间维度——短期运输瓶颈vs中长期产量

**当前不能确认的事项**

- 不能将近远月价差（~$25）等同于单一"战争溢价"——该差值可能包含：短期运输瓶颈定价、供应中断风险溢价、政策响应不确定性定价、波动率期限结构溢价
- 不能确定远月价格是否代表"冲突后均衡"——远月流动性低，定价效率较差

**反证/削弱该判断的数据**

- 如果近远月价差快速收窄（M1-M6 < $5），则短期供应紧张信号显著弱化
- 如果远月价格也开始上涨（如$80+），可能反映市场对长期供给格局变化的重定价

### 2.3 航运数据

**观察到的事实** [证据层级: A（通行量）] [数据截至: 2026-03-17]

- 霍尔木兹海峡：7日均值 1.1艘 vs 90日均值 39.9艘（周环比 -97.6%）
- 曼德海峡：7日均值 11.7艘 vs 90日均值 11.5艘（周环比 -7.9%，基本正常）
- 苏伊士运河：7日均值 16.7艘 vs 90日均值 14.8艘（持平）
- 马六甲海峡：7日均值 81.3艘 vs 90日均值 83.0艘（持平）

**与这些事实一致的解释**

- 解释1：霍尔木兹通行量显著受阻，通行已大幅减少
- 解释2：AIS统计口径可能导致读数失真——仅计入某类船型或某区域、部分油轮关闭AIS转发器规避冲突、数据源覆盖延迟
- 解释3：其他三个咽喉基本正常，中断可能集中在波斯湾出口通道

**当前不能确认的事项**

- 实际中断桶数——AIS数据无法直接转换为贸易流量（需Kpler/Vortexa商业数据）
- 通行限制是否针对特定国家（伊朗外长曾声明"仅对敌人关闭"）
- 航道关闭程度是"部分受限"还是"接近完全封锁"

**数据可信度注记**

⚠️ 霍尔木兹 -97.6% 为极端读数。如果全球约25%海运石油真的接近完全中断，理论上油价应远超$120-150+。当前WTI $98与完全封锁之间存在缺口。这意味着：AIS数据可能高估封锁程度，或替代路线/SPR释放正在部分缓解影响，或市场尚未完全定价最坏情景。

**反证/削弱该判断的数据**

- 如果霍尔木兹7日均值回升至>10艘，则通行正在恢复
- 如果曼德海峡同步骤降，则中东出口全面受阻的假设强化
- 如果苏伊士流量大幅上升，可能反映替代路线正在发挥作用

- STNG (成品油)：$68.42，5日 -9.1%

**与这些事实一致的解释**

- 解释1：油轮板块股价未体现"运费受益"的直觉方向，与"货源减少效应大于运费上涨效应"一致
- 解释2：也可能反映市场恐慌情绪下的无差别抛售、流动性收缩、或板块轮动
- 解释3：成品油轮STNG跌幅最大（-9.1%），与成品油运输受影响更大一致，但样本量太小（4只股票）不能确认

- 不能仅凭股价走势确认"无货可运"是主导原因

- 无法区分"基本面定价"和"恐慌抛售"的各自贡献

**反证/削弱该判断的数据**

- 如果油轮股企稳反弹而航运通行量仍低，则"无货可运"叙事可信度下降
- 如果整体A股/美股能源板块同步暴跌，则可能是系统性抛售而非油轮特异信号

- 解释2：净多仓处于26%分位，与"做多空间尚未被大规模使用"一致

**反证/削弱该判断的数据**

- 如果下一期CFTC显示净多增加主要由新多入场驱动（而非继续空头回补），则上涨持续性假设增强

- 解释2：OVX 119处于极端水平，与严重地缘事件冲击一致
- 解释3：在事件驱动的市场中，金融条件的影响通常被供给冲击主导，但不能排除宏观因素对价格天花板的抑制作用

- OVX极端值的具体成因——可能包含对冲需求飙升、做市商撤出流动性、期权市场结构性变化

**反证/削弱该判断的数据**

- 如果OVX快速回落至<60，表明市场恐慌情绪缓解
- 如果DXY走弱，则消除强美元逆风

## 三、假设集与验证

### 假设A：当前价格中含显著地缘风险成分

**支持证据** [主要为A层]
- OVX 119.02处于99.4%历史分位（A层）
- 期货近远月价差$25+（A层）
- 霍尔木兹通行量骤降（A层，但精度存疑）
- Brent溢价走阔至-$6.41（A层）

**反证据**
- 当前价格如果仅$98而实际中断已达60-70%，说明地缘溢价可能已部分被SPR释放/替代路线对冲
- STEO显示中长期供过于求+2.56 mb/d（B层），基本面本身不支持$98价位
- 无法精确量化地缘成分与基本面成分各占多少——近远月价差~$25可能包含运输瓶颈、风险溢价、波动率期限结构等多项

**当前证据强度**：中-高（4个A层源方向一致，但"地缘主导"中的"主导"程度无法精确量化）

**下一步验证条件**
- 如果霍尔木兹通行量恢复（>10艘/日）而价格未显著下跌 → 地缘成分可能被高估
- 如果OVX回落至<60而价格维持$90+ → 基本面支撑可能比预期更强
- 如果近远月价差收窄至<$10 → 短期供应紧张信号显著减弱

### 假设B：实物供给通道已显著受阻

**支持证据** [混合层级]
- 霍尔木兹AIS通行量-97.6%（A层，但精度存疑）
- 其他三个咽喉基本正常（A层）——中断集中在波斯湾出口
- Brent溢价持续走阔（A层）
- 油轮股集体下跌-7.1%（C层辅助信号，方向一致但不能单独确认）

**反证据**
- AIS -97.6%与WTI $98之间存在定价缺口——完全封锁对应的理论价格远高于$98
- 实际中断规模可能低于AIS数据显示——需商业航运数据验证
- 替代路线（Red Sea出口、Kirkuk-Turkey管道）可能已部分缓解影响

**当前证据强度**：中等（A层航运数据方向明确，但绝对值精度存疑）

**下一步验证条件**
- 需要商业航运数据（Kpler/Vortexa）确认实际桶数影响
- 如果Brent-WTI价差开始收窄 → 国际海运原油供给恢复信号
- 如果曼德海峡通行也大幅下降 → 中东出口双通道受阻，中断升级

### 假设C：近期上涨中空头回补贡献高于新多入场

**支持证据** [A层但时效有限]
- 空头减少-21,074张远大于多头增加+2,663张（A层，截至3/10）
- 净多仓26%分位——大量做多空间尚未使用（A层）

**反证据**

### 代理变量 vs 直接证据的覆盖差距

当前报告中使用了以下C层代理变量：
- **油轮股**：用作航运中断的辅助信号，但不能替代实际贸易流数据
